# Laptop Price Predictor

**Laptop Price Predictor** is a machine learning web application that predicts the price of a laptop (in euros) based on its hardware specifications. The model is trained using a **Random Forest Regressor**, providing accurate and efficient predictions through a simple and user-friendly web interface.

## Features
- Predicts laptop prices based on specifications such as **RAM, CPU, GPU, Storage**, and more.
- Utilizes **one hot encoding** for handling categorical data.
- Built with a **lightweight Flask backend** for fast performance.
- Simple **web interface** built with HTML, CSS, and Bootstrap.
- Trained on a real world laptop dataset for reliable predictions.

## Machine Learning Model
- **Algorithm:** RandomForestRegressor  
- **Dataset:** Laptop hardware specifications and prices (in euros)  
- **Preprocessing:** One hot encoding for categorical features  
- **Libraries Used:** scikit-learn, pandas, numpy, pickle  

## Tech Stack
**Frontend:** Streamlit 
**Backend:** Flask (Python)  
**Machine Learning:** scikit-learn, pandas, numpy, pickle  

## Output
Displays the **predicted laptop price (in euros)** based on the entered specifications.
